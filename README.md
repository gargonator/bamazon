## Bamazon

In order to...

Intialize the Bamazon store: use `bamazonInitialize.js`

Buy products from Bamazon: use `bamazonCustomer.js`

Manage Bamazon (view products, low inventory, add stock, add new products): use `bamazonManager.js`
